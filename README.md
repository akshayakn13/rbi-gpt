# 🏦 RBI GPT

A RAG-based AI assistant that answers questions about RBI Master Circulars. Built for bankers, compliance officers, and general public.

## What it does
- Answers questions about RBI Master Circulars instantly
- Cites the exact circular the answer comes from
- Simple explanation mode for non-bankers

## Tech Stack
- Streamlit, ChromaDB, SentenceTransformers, Gemini 2.5 Flash
